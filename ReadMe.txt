Database : 
data content file : OnlineStore-data-dump.sql
To import database content:
login in mysql and create new database (for example: create database OnlineStore; )
After created new database, exit mysql terminal, use command :  mysql -u username -p new_database < OnlineStore-data-dump.sql


PHP : 
recommend version:  7.2  or 7.4

PHP project notes:
modify database connection information  in OnlineStore/Models/database_con.php
